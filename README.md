# Odin Recipes

This is a basic recipe website built as part of The Odin Project curriculum.
It demonstrates foundational HTML skills, including working with boilerplates, links, lists, and images. 

## Technology Used

- HTML
- Git
- GitHub
- WSL2 
- Markdown
- VSC
